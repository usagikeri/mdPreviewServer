# Markdown Preview Server by Flask

---

Markdown-file convert html5.

# Description

Markdown-file convert html5.
Preview local server (127.0.0.1:5000)

# Requirement
* flask  
```
$pip install flask
```
* pandoc

## osX
```
$brew install pandoc
```

## Ubuntu
```
sudo apt-get install pandoc
```

# Usage
Put Markdown-file in md-directory.

.```
$make run
```
